BuildHub

This site intends to present information about the products/projects in our Organization.

The information is obtained from Jenkins/SonarCube and other tools used by the builds and centralized in an easy to read format.

    The builds are grouped per product. The site currently contains the following groups:
  • Foundation
  • Order Management
  • Provisioning
  • CPQ

  • The versions of the projects are currently coming from a configuration file. The intention was to get the version from the project dynamically but I didn't find a way to do so.
